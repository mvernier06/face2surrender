Welcome to face2surrender’s documentation!¶

face2surrender is a Python application, using the CelebA database to identify a criminel. It is using a variational autoencoder and a genetic algorithm in order to propose some pictures of faces. We use several selection of pictures to create new faces in order to find a face that looks like the criminal we are looking for. It provides a simple and intuitive interface to do so.

Check out the Usage section for further information, including how to install the project.

For more information on the autoencoder, check out the autoencoder section of the Usage (you’ll have information on the data too).

And finally, you can check how we generated new faces with the genetic-algorithm on the Usage.

Note

This project is under active development.
